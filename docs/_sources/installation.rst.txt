============
Installation
============


Stable release
--------------

To install BioNumPy Assembly, run this command in your terminal:

.. code-block:: console

    $ pip install bnp_assembly

or for the newest:

.. code-block:: console

    $ pip install git://github.com/knutdrand/bnp_assembly


If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for BioNumPy Assembly can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/knutdrand/bnp_assembly

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install -e .


.. _Github repo: https://github.com/knutdrand/bnp_assembly
.. _tarball: https://github.com/knutdrand/bnp_assembly/tarball/master
